# lyceum-application

How to run:

1. Make 'docker-compose up' in root dir
2. Connect to MySql DB: username = "root", password = "password", url = "jdbc:mysql://127.0.0.1:3307/"
3. Run creation_script.sql from lyceum-application/database
4. (optional) run scripts from lyceum-application/database/scripts for filling DB
5. Executing gradle 'run' task 
6. Application run on port 8088
